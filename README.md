# Autonomous Desktop

Este projeto foi desenvolvido no 5º semestre da faculdade, com o objetivo de atender a uma demanda real de uma comunidade externa à instituição de ensino.

A solução proposta consiste em um sistema completo de gestão de freelancers, composto por uma aplicação desktop para administração, uma API responsável pela centralização dos dados e um site web para acesso e interação dos usuários.
